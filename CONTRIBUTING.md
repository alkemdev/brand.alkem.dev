# Contributing

1. Clone, then `cd site && npm install`
2. Edit sources in `src/` or `reference/` (not `out/`)
3. Run `just build` to regenerate; `just dev` to preview
4. Use `just` recipes; don’t hand-edit generated files

**Where to edit:** tokens → `src/palette/tokens.json`; site → `site/`; logo → `reference/logo/`; 3D flask (JS) → `site/src/lib/flask-3d/`.
